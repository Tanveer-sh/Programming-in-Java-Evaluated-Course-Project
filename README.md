# Java Strategy Game: Digital Tic-Tac-Toe



### Project Overview:



This project is a Java-based Tic-Tac-Toe game. It's a tool for fun and socializing. Unlike games that use a lot of data this one works on any computer with Java.



#### Features:



* Interactive Command Line Interface: You play on a numbered grid. It's easy to use.



* Error Handling: The game catches mistakes when you enter something thats not a number or, out of range.



* Game Results: The game saves results and times to a file called game\_history.txt.



* Winning Logic: The game checks for 8 winning ways and draws.



#### Concepts Used:



* Object-Oriented Logic: The game code is organized and clean.



* File Input/Output: The game uses FileWriter to save data.



* Data Structures: The game uses arrays to manage the board.



* Exception Handling: The game handles user input mistakes.



#### Execution:



* Get the Java Development Kit (JDK).



* Download TicTacToe.java.



* Open your terminal. Command prompt.



* Compile the code:



javac TicTacToe.java



Run the game. Play Tic-Tac-Toe!
